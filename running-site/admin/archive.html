📥 Archive All Rankings

Snapshots every event in the live rankings as a new week entry. Uses the Week ID and Week Label set on each event in the CMS, or falls back to a date-based ID.

🎯 Load Event Into Week

Pulls the current live rankings for one event into a specific week ID. Use this when you've already added the week ID to the archive index and just need to fill it with data.

Must match the ID you added in Rankings Archive.
